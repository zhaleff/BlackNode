# BlackNode

Arch Linux + Hyprland dotfiles with Material You theming, an updated version of HyprCraft

- Clear structure, well-organised directories.
- No obscure hacks — readable configuration files.
- Modular by design: remove or add components without breaking the rest.

For documentation, see `INSTALLATION.md`, `KEYBINDS.md`, `ROFI.md`, `PACKAGES.md`, and `MODULES.md`.

Blacknode is a clean, classic environment built using classic elements—without QuickShell—allowing it to run on modest hardware.

# Why?

The project structure is simple and easy to understand; users who want to create their own dotfiles can use mine as a base. That is the reason for the simplicity—I want them to be able to modify and create their own files.

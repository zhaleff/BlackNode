# Waybar

Three styles, all using colours generated by matugen.

## Classic (default)

Full-featured bar. Left: workspaces, media controls (MPRIS + previous/pause/next), system tray (backlight, volume, network, power, battery), package updates (pacman + AUR). Right: window title, idle inhibitor, clock, system resources (CPU, temperature, RAM), user info.

## Minimal

Three modules: workspaces (left), clock (right), system tray (right). Zero clutter.

## Hacking

Designed for CTF and penetration testing. Includes VPN status and target IP module. Currently under development.

## Switching

Edit `~/.config/waybar/style.css` and restart Waybar, or use `~/.local/bin/switch-waybar-style.sh`.

## Custom Modules

| Module | Purpose | Action |
|--------|---------|--------|
| `custom/pacman` | Official package updates | Click opens `sudo pacman -Syu` |
| `custom/pkg-aur` | AUR package updates | Informational only |
| `custom/power` | Power menu | Click opens wlogout |
| `custom/user` | Username display | Click sends `whoami@hostname` notification |
| `custom/previous` | Previous track | Click sends `playerctl previous` |
| `custom/pause` | Play/pause toggle | Click toggles playback |
| `custom/next` | Next track | Click sends `playerctl next` |

## Keybindings for Waybar

- `SUPER + SHIFT + W` — toggle Waybar visibility (if bound)
